================================================================================
QUICK SETUP: Hugging Face Speech-to-Text
================================================================================

STEP 1: Get Hugging Face API Key
---------------------------------
1. Go to: https://huggingface.co/settings/tokens
2. Click "New token"
3. Name it: "BuddyBud Speech-to-Text"
4. Select permissions: "Inference Providers" (for API access)
5. Click "Generate token"
6. Copy the token (starts with "hf_...")

STEP 2: Add to .env File
-------------------------
1. Open your .env file (or copy from .env.example)
2. Add this line:
   
   HUGGINGFACE_API_KEY=hf_paste_your_actual_key_here

3. Save the file

STEP 3: Install Dependencies
-----------------------------
Run in terminal:
   pip install requests==2.31.0

STEP 4: Run Migrations (ALREADY DONE)
--------------------------------------
   python manage.py migrate

STEP 5: Test It!
----------------
1. Start server: python manage.py runserver
2. Go to student flow
3. Enter homework code
4. Upload answers
5. Wait for AI analysis
6. Start interview
7. RECORD VIDEO WITH AUDIO (speak clearly!)
8. Complete interview
9. Wait ~30-60 seconds for transcription
10. View results with personalized feedback!

================================================================================
TROUBLESHOOTING
================================================================================

Error: "HUGGINGFACE_API_KEY not configured"
→ Solution: Add key to .env file (see Step 2)

Error: "No module named 'requests'"
→ Solution: pip install requests==2.31.0

Error: "Empty transcription returned"
→ Solution: Ensure video has audio and you spoke clearly

Error: "Timeout"
→ Solution: Record shorter responses (<5 minutes total)

Error: "No interview recording found"
→ Solution: Ensure video recording saved properly (check browser console)
